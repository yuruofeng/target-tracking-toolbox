# Extended-Target Tracking

Extended-target algorithms live under `tracking.extended`.

- `tracking.extended.ggiw`: GGIW-PHD filter and extended measurement utilities.
- `tracking.extended.starconvex`: star-convex shape tracker and generators.
- `tracking.extended.phd`: extended-target PHD filter and measurement partition.
- `tracking.extended.utils`: utilities local to the extended-target module.

Examples:

```matlab
run('examples/matlab/demo_extended_ggiw.m');
run('examples/matlab/demo_extended_starconvex.m');
```
